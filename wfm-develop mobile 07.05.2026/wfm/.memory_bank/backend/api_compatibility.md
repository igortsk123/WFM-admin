# API Совместимость — временные эндпоинты

Этот файл фиксирует эндпоинты, которые существуют в двух версиях ради обратной совместимости с мобильными клиентами.

**Правило:** старая версия удаляется после того, как все клиенты (iOS/Android) обновились до версии выше указанного порога.

---

## GET /tasks/list/filters

| | |
|---|---|
| **Старая версия** | `GET /tasks/list/filters` |
| **Новая версия** | `GET /tasks/list/filters/v2` |
| **Причина** | Новая версия добавляет фильтр «Сотрудники» (assignee_ids) и меняет порядок групп: work_types → assignees → zones |
| **Старая версия поддерживает** | iOS ≤ 1.0.4, Android ≤ 1.0.4 |
| **Условие удаления старой** | Все клиенты на prod обновлены выше указанных версий |
| **Статус** | В работе |

## GET /tasks/list

| | |
|---|---|
| **Старая версия** | `GET /tasks/list` |
| **Новая версия** | `GET /tasks/list/v2` |
| **Причина** | Новая версия применяет zone_ids и work_type_ids как пересечение (AND), старая — как объединение (OR) |
| **Старая версия поддерживает** | iOS ≤ 1.0.4, Android ≤ 1.0.4 |
| **Условие удаления старой** | Все клиенты на prod обновлены выше указанных версий |
| **Статус** | В работе |
