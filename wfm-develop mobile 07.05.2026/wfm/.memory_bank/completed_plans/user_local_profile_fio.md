# План: Локальное хранение ФИО пользователя

**Статус:** Выполнено
**Создан:** 2026-04-08
**Последнее обновление:** 2026-04-08

## Цель

Добавить поля `first_name`, `last_name`, `middle_name` в таблицу `users` как локальный источник ФИО.

**Проблема:** При теневой регистрации партнёрских пользователей в SSO поля ФИО пустые. LAMA для партнёров неприменима. В итоге пользователь отображается без имени и фамилии.

**Решение:** Хранить ФИО в `users` — задаётся при создании/приглашении пользователя. Новый приоритет при сборке ответа `/me`:
```
LAMA cache → users.first_name/last_name/middle_name → SSO cache
```
