# Web Admin Tech Stack (рекомендации)

## Цель
Быстро собирать админские панели без перегрузки команды.

---

## Рекомендованный стек

### 1. **Next.js 15 (App Router)**
- Универсальный рендеринг
- Быстрое построение форм
- Открыт для интеграций

### 2. **TypeScript**
Базовый выбор.

### 3. **shadcn/ui**
- Очень быстро собирать формы
- Легко стилизовать
- Один из лучших design-kit в 2024–2025

### 4. **React Query**
Работа с API приложения.

### 5. **Zod**
Валидация форм.

### 6. **Posthog или Sentry**
Аналитика поведения и ошибок.

---

## Результат
Админка получается:
- удобная,
- масштабируемая,
- легко поддерживаемая,
- со скоростью разработки как у стартапа.
