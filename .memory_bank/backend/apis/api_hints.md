# API Подсказок (Hints)

**Base URL:** `/tasks/hints/`
**Сервис:** `svc_tasks`

## Назначение

Подсказки — краткие, отредактированные вручную тексты, помогающие работнику выполнить задачу в конкретной зоне. Привязаны к комбинации (work_type_id + zone_id). Одной паре может соответствовать несколько подсказок.

## Связь с operations

Таблица `operations` содержит поля `hint_1..hint_6` — сырые подсказки из LAMA, описывающие шаги операции. Они служат **исходным материалом** для формирования подсказок в `hints`. Прямой FK-связи между таблицами нет намеренно: подсказки в `hints` редактируются вручную и могут существенно отличаться от исходника.

## Эндпоинты

| Метод | Путь | Доступ | Описание |
|-------|------|--------|----------|
| GET | `/hints?work_type_id=X&zone_id=Y` | Все авторизованные | Список подсказок, сортировка по id |
| POST | `/hints` | MANAGER | Создать подсказку |
| PATCH | `/hints/{hint_id}` | MANAGER | Обновить текст подсказки |
| DELETE | `/hints/{hint_id}` | MANAGER | Удалить подсказку |

## Модель

```json
{
  "id": 1,
  "work_type_id": 1,
  "zone_id": 2,
  "text": "Расставляй товары лицом к покупателю"
}
```

## Формат ответа

Все эндпоинты возвращают стандартный `ApiResponse`. GET возвращает `{ "hints": [...] }`.
