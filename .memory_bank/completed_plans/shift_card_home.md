## Цель
Добавить карточку смены на главный экран (HomeView/HomeScreen) с отображением данных о текущей смене пользователя. Использовать дизайн из Figma с двумя состояниями: карточка с данными и заглушка при отсутствии данных.

## Контекст
- Запрос текущей смены уже выполняется в UserManager (`currentShift`)
- Дизайн компонента карточки: https://www.figma.com/design/J9crsNS4wIszpILzWvnY2T/Smart-employee?node-id=1777-78712
- Использовать только токены цветов из WFMColors/WfmColors (не на глаз!)

## Состояния карточки смены

Компонент имеет **6 состояний** (варианты):

1. **NEW** — Новая смена (ещё не началась)
   - Бейдж с должностью (фиолетовый)
   - Время смены (8:00 - 20:00)
   - Статус: "Начнется через X мин" (серый)
   - Адрес магазина + длительность смены
   - План дня с задачами (не зачёркнутые)
   - Кнопка "Открыть смену" (primary)

2. **IN_PROGRESS** — Смена в процессе
   - Аналогично NEW, но:
   - Задачи зачёркнуты (выполнены)
   - Кнопка "Взять новую задачу" (secondary)

3. **DELAY** — Опаздывает на смену
   - Аналогично NEW, но:
   - Статус: "Вы опаздываете на X мин" (КРАСНЫЙ цвет - cardTextError)
   - Кнопка "Открыть смену" (primary)

4. **DONE** — Смена закрыта
   - Время смены
   - Статус: "Смена закрыта" (серый)
   - Адрес + длительность
   - План дня с зачёркнутыми задачами
   - Кнопка "Открыть смену" (primary) - для следующей смены

5. **NO_DATA** — Данные не загрузились
   - Иконка обновления (фиолетовый круг)
   - Заголовок: "Данные не загрузились"
   - Подпись: "Попробуйте обновить страницу"
   - Кнопка "Обновить" (secondary)

6. **EMPTY** — Нет задач
   - Иконка info (фиолетовый круг)
   - Заголовок: "У вас нет задач"
   - Подпись: "Ожидайте назначения задач или обратитесь к директору"
   - Кнопка "Взять новую задачу" (secondary)

## Технические детали

### Токены цветов (важно!)
- **Фон карточки**: `surfaceSecondary` (белый)
- **Фон секции**: `cardSurfase` (#f5f5fc)
- **Текст основной**: `cardTextPrimary` (#373742)
- **Текст вторичный**: `cardTextSecondary` (#5e5e6d)
- **Текст ошибки (опоздание)**: `cardTextError` (красный) - ВАЖНО для состояния DELAY
- **Бордер**: `borderSurfaceSecondary` (#eeeef8)
- **Бейдж фиолетовый**: `badgeBrandBgLight` (#efe8ff фон), `badgeBrandText` (#6738dd текст)
- **Кнопка primary**: `buttonPrimaryBgDefault` (#6738dd), `buttonPrimaryTextDefault` (white)
- **Кнопка secondary**: `buttonSecondaryBgDefault` (#efe8ff), `buttonSecondaryTextDefault` (#6738dd)

### Типографика
- **Время смены**: Headline24Bold
- **Статус**: Caption12Medium
- **Адрес/длительность**: Caption12Medium
- **Заголовок заглушки**: Headline18Bold
- **Подпись заглушки**: Body16Regular

### Отступы
- **Горизонтальный padding контента**: `xl` (16dp)
- **Padding внутри карточки**: `m` (12dp)
- **Gap между элементами**: `s` (8dp), `xs` (6dp)
- **Радиус карточки**: `xl` (16dp)
