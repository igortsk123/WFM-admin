# Планы других платформ (mobile / backend)

Эти планы унаследованы от старого общего memory bank в `WFM/.memory_bank/`. Они **не для admin**, а для команд mobile (iOS/Android) и backend (FastAPI).

Лежат здесь, чтобы при старте сессии Claude (admin scope) не подхватывал их как «активные планы admin».
