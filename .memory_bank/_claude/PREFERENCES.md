# User preferences (workflow + style)

## Workflow

- Не дублирую системную Claude memory — пишу всё в `.memory_bank/` (репо)
- НЕ деплою руками — только через CI/CD (см. [DEPLOY.md](./DEPLOY.md))
- После каждого мержа в main → автодеплой сам
- Перед запуском V0 промпта — foundation от меня смерджен в main

## Терминология (UI + код)

| Не использовать | Использовать |
|---|---|
| операции (UI) | подзадачи (subtasks) |
| Привилегии | Зоны |
| Грейд | Разряд |
| Архив задач | Задачи |
| Дополнительная (тип задачи) | удалена, есть Плановая + Бонусная |
| Group (для product) | Категория (товарная) ≠ Группа (work_type.group) |

## Семантические токены

**Только** `bg-primary`, `bg-success`, `bg-warning`, `bg-destructive`, `bg-muted`, `bg-info`.

❌ НИКОГДА `bg-blue-500`, `bg-emerald-50`, hex-цвета.

## Язык

- Russian UI labels, English code identifiers
- Документация на русском
- Стиль РУ:
  - Тире (—) использовать редко (раздражает как признак ИИ)
  - Естественный язык, как человек написал бы
  - Запятая, точка, союз — чаще выглядят естественнее
  - В EN em-dash — норма (не затрагивается)

## Тон в общении

- Короткие ответы, конкретно по делу
- Без избыточных summary в конце
- Без emoji (если user не просит)
- Не извиняться попусту, не повторять «отлично/принято/конечно»

## Bug reports от user

Формат «Экран X, проблема Y, ожидаю Z». Я разношу:
1. **Бизнес-нелогичности** → V0 patch-чаты
2. **Визуальные баги** → точечный fix мной (если небольшой)
3. **Лишний функционал** → удалить
4. **Технические недочёты** → я сам

## Отдельная папка для скринов

User кидает скрины в чат — они складываются в `.memory_bank/screens/` (имя оригинальное от телеграма или семантическое).

## Декомпозиция кода

V0 генерит монолит (page.tsx 500-800 строк) — это нормально. Декомпозиция и причёска — моя работа в Этапе 3 проекта (Cleanup → Decomposition → Linking → Documentation).

## Подработка (важно — концепт)

Подработка ≠ FREELANCE. Это **тот же STAFF сотрудник** в свой выходной вышел в **другой магазин** того же юрлица. Должность та же, ЗП штатная, легально.

В коде: `Shift.shift_kind = 'SUBSTITUTE'` + `home_store_id` (родной магазин).

На pie chart циклограммы канал PART_TIME = shift_kind=SUBSTITUTE (не employment_type).
