### 10.1. Тип задачи: убрать "Дополнительная"

**Файл:** `components/features/tasks/task-form.tsx`

User: «Тип задачи может быть плановая и бонусная. Дополнительной задачи быть не может.»

Сейчас RadioGroup `type`:

- Дополнительная (ADDITIONAL) — **удалить**

### 10.2. Автоподставление магазина при заходе из stores/[id]

**Файл:** task-form.tsx + page wrapper.

User: «Если мы провалились из магазина, надо автоматически подставлять магазин.»

Логика:
- При переходе со страницы магазина в task-form — URL должен содержать `?store_id=N`
- Page wrapper читает search params (можно через `useSearchParams()` если client) и передаёт `defaultStoreId` в form
- Form: `useForm` initial value для store field — этот id

В Stores Detail (MY-PLAN-6.7) кнопка "Создать задачу" должна быть `<Link href={`/tasks/new?store_id=${id}`}>`.
